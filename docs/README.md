## Terminology

Use:

- VLESS;
- Telegram proxy / MTG;
- Telegram link;
- DoubleHop Mode;
- `sudo <prefix>-xpam`;
- `sudo <prefix>-links`.

Public docs must not contain real domains, IP addresses, connection links, UUIDs, tokens or local operator paths.
